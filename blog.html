Q:What is the Document Object Model (DOM)?

Answer:The Document Object Model (DOM) is a programming interface provided by web browsers that allows developers to interact with and manipulate the structure, content, and style of web documents (usually HTML and XML). In simpler terms, it is the structured representation of a web page that enables scripts, such as JavaScript, to access and modify the content dynamically.

Q:How do you select an element from the DOM?

Answer: getElementById

Q: What is event delegation in the context of the DOM, and why is it useful?

Answer:Event delegation is a technique in the context of the DOM where a single event listener is attached to a parent element, and it handles events triggered by its child elements.
