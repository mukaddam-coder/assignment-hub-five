Important FAQs!

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface provided by web browsers that allows developers to interact with and manipulate the structure, content, and style of web documents (usually HTML and XML).

How do you select an element from the DOM?

Selecting by ID: getElementById

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the context of the DOM where a single event listener is attached to a parent element, and it handles events triggered by its child elements.
